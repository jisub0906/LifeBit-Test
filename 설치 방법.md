### 0. 개발 언어 통일
```bash
java sdk-21
python 3.13.5
nodejs latest lts
pnpm latest lts
```

### 1. Docker 환경 설정(데이터베이스) 최초 1번만!
```bash
docker-compose up -d
```

### 2. 프로젝트 루트 폴더에서 진행
```bash
pnpm install
```

### 3. AI API (FastAPI) 설정
```bash
# AI API 디렉토리로 이동
cd apps/ai-api-fastapi

# Python 가상환경 생성 및 활성화
python -m venv venv

# Windows
venv\Scripts\activate
# macOS/Linux
source venv/bin/activate

# 의존성 설치
pip install -r requirements.txt
```

### 4. 프로젝트 루트 폴더에서 진행
```bash
# vite 디렉토리로 이동
cd apps/frontend-vite

pnpm install
```
